Download Extension ZIP

Installation Instructions:

  1. Click the "Download Extension ZIP" button above
  2. Extract the downloaded ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked"
  6. Select the folder where you extracted the ZIP file
  7. The extension will be installed and ready to use!
Note: After installation, you'll see the Aviator Predictor icon in your Chrome toolbar. Click it to start using the predictor!
